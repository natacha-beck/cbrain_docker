What is Docker?

  • Motto: Build, Ship, Run
  • Open platform for distributed applications
  • Allow you to package your app with all its dependencies

What is Docker?

  • A container contain a piece of software wit everything it need to run
  • Code, system tools, libraries...
  • So the software should always run the same (not depending of the environment)

Difference between VM and Docker?

  • Containers include app and all the dependencies
  • BUT share the kernel between each other
  • So it is light, more portable and efficient than VM
  • It is supported by Compute Canada! (On Guillimin for the moment,Mammouth soon)

Limitations?

  • Run on Unix system
  • On Mac and Windows it require VM!
  • Painfull to use on Mac (own experience)

What is DockerHub?

  • Similar to Github but for Docker container sharing
  • Public/private repo
  • Collaborators, can have some privilege (example: push new version)
